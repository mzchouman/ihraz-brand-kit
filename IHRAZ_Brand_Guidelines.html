02

The Convergence Mark

Visual symbol of intelligent achievement through unified intelligence streams

Concept

The Convergence Mark represents five streams of AI intelligence converging at a singular point of achievement. Each arc symbolizes a different intelligence stream—data analysis, predictive modeling, resource optimization, goal alignment, and performance measurement—all flowing together to enable organizational success.

Construction

The mark is mathematically constructed from five Bézier curves with precise control points, creating a harmonious convergence. A central apex point with radiating glow represents the moment of achievement. Nodes along each arc symbolize data intersection points.

Usage Guidelines

✓ DO

  • ✓ Use on dark backgrounds (dark theme preferred)
  • ✓ Maintain clear space (minimum 40px on all sides)
  • ✓ Use official emerald colors only
  • ✓ Scale proportionally to any size
  • ✓ Allow breathing room in layouts
  • ✓ Ensure minimum 24px display width

✗ DON'T

  • ✗ Rotate or distort the mark
  • ✗ Change color values or apply effects
  • ✗ Place on competing busy backgrounds
  • ✗ Use drop shadows or outer glows
  • ✗ Reshape or simplify the arcs
  • ✗ Apply gradients beyond the built-in rendering

Minimum Clear Space: The mark requires a minimum buffer of 40 pixels on all sides. No other design elements should intrude into this space.

Color Usage Rules

Dark Neutrals (Void, Deep, Surface)
Primary backgrounds, layering, depth creation. Establish visual hierarchy and containment.

Emerald Spectrum
Brand accent color. Use Emerald 400-300 for interactive elements, Emerald 600-700 for backgrounds.

Text & Borders
Use Text Mid for secondary content, Text Light for primary. Border color for dividers and subtle edges.

Contrast & Accessibility
All text meets WCAG AA minimum contrast. Emerald 300 text on dark backgrounds achieves 4.5:1 contrast.

Common guidelines ensuring consistent brand application

  • ✓ Use on contrasting backgrounds

  • ✓ Use official color palette

✗ DON'T

  • ✗ Rotate or distort the mark
  • ✗ Apply drop shadows or effects
  • ✗ Change colors or apply gradients
  • ✗ Place on busy backgrounds
  • ✗ Use smaller than 24px width

Color Usage

✓ DO

  • ✓ Use Emerald 400-300 for actions
  • ✓ Maintain high contrast for text
  • ✓ Use dark backgrounds primarily
  • ✓ Reference hex values exactly
  • ✓ Test for color accessibility

✗ DON'T

  • ✗ Approximate colors visually
  • ✗ Create new brand colors
  • ✗ Use very light text on light backgrounds
  • ✗ Mix with unrelated color palettes
  • ✗ Reduce opacity of brand colors

Typography

✓ DO

  • ✓ Use Syne for headlines
  • ✓ Use Cairo for Arabic content
  • ✓ Use Space Grotesk for body
  • ✓ Maintain specified line heights
  • ✓ Respect font weight recommendations

✗ DON'T

  • ✗ Mix multiple serif fonts
  • ✗ Use body font for headlines
  • ✗ Apply text transformations beyond spec
  • ✗ Reduce line height for aesthetics
  • ✗ Substitute fonts arbitrarily
