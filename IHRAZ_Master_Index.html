How to Share This Brand Kit

This index is fully self-contained. To share with your team, designers, or external partners:

1. Copy the entire IHRAZ_Identity folder as-is.

2. Send via file share, Google Drive, Dropbox, or any file transfer method.

3. The recipient opens IHRAZ_Master_Index.html in any browser.

4. All links are relative — every asset opens or downloads directly from the same folder.

No server, no hosting, no dependencies. Just the folder and a browser.
